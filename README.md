### Additional description about the project and its features
- THW logo
- A top nav bar
- A sticky bottom nav
- Sections and article elements with contents
- A grid layouts with images and lists
- Footer top and footer bottom
- Responsive on different viewports

## Built With

- HTML
- CSS
- Media queries
